How to Play

The dealer draws one card at a time. Place each card into one of three sacred rows — God Loved, He Gave, or We Believe. Each row must add up to exactly 16. Complete all three rows to reveal John 3:16.

Discarding: Send a card to the Temptation pile instead of placing it. Too many discards ends the game.

  • Easy: 12 discard limit
  • Medium: 8 discard limit
  • Hard: 5 discard limit

Grace Tokens: If a row goes over 16, a Grace Token removes the offending card and saves the row. No tokens left = game over. Each bust costs −50 score.

Grace Cards: Special cards in the deck that instantly give you +1 Grace Token when drawn. They auto-advance to the next card.

Card Values:

  • Easy — Alpha(A)=1, Disciple(J)=1, Faithful(Q)=2, King(K)=3, Numbers=face value
  • Medium/Hard — Alpha(A)=1, Disciple(J)=11, Faithful(Q)=12, King(K)=13, Numbers=face value

Hard Mode: Each row must contain at least 3 cards before it can lock at 16. If a row reaches 16 with fewer than 3 cards, play continues and bust rules apply.

Easy Mode Preview: The next card in the deck peeks out behind the current card.

Scoring: Start at 1000. Lose −10 per card drawn, −25 per discard, −50 per Grace Token used on a bust. Win bonuses: +50 Easy / +100 Medium / +200 Hard, +100 for zero discards, +100 for zero Grace Tokens used.
